# Recipe wizard a11y v3 polish

## Summary

Nine a11y + polish findings on the recipe wizard. The v2 polish from commit `de90151` landed most tap-target and announcement work; these are the consistency gaps and SR-feedback items that drifted or were missed.

## Findings (cross-ref `docs/audits/2026-04-18-full.md`)

### Tap targets

- **M30** — `PreviewStep.EditButton` (6 uses) has `fontSize: 13` + `hitSlop={8}` → ~29-31px effective. WCAG 2.5.5 requires 44×44. Sibling `InstructionsStep` reorder was fixed to 44×44 in prior M19 — this drifted.
- **M31** — `TimeServingsStep.stepperButton` is `width: 36, height: 36` (visible). iOS HIG + WCAG 2.5.8 recommend 44×44 visible.
- **M32** — Servings stepper has no `accessibilityValue` / `accessibilityRole="adjustable"` — VoiceOver users pressing +/− get no value announcement.

### Screen-reader feedback

- **L12** — `InstructionsStep.handleMove` fires haptic but no `AccessibilityInfo.announceForAccessibility` (sibling `handleRemove` does).
- **L13** — `WizardShell.editFromPreview` jumps steps without SR announcement ("Step N of 7").
- **L16** — `SearchFilterSheet` sliders fire only `onSlidingComplete` — no live SR feedback during drag.

### Animation + polish

- **L11** — `StandaloneSkeletonBox` fallback (outside SkeletonProvider) still creates `useAnimatedStyle` under reducedMotion (held constant). Minor perf drift.
- **L14** — Wizard slide animation direction is "forward" when returning to Preview from edit-step (semantically feels backward).
- **L15** — `IngredientsStep.inputRefs` Map is wired but never read — dead code or unimplemented auto-focus. Either wire up auto-focus or delete.

## Acceptance Criteria

- [ ] PreviewStep edit buttons ≥ 44×44
- [ ] TimeServingsStep stepper ≥ 44×44 visible
- [ ] Servings stepper has `accessibilityValue` + role
- [ ] Reorder action announces result
- [ ] editFromPreview announces target step
- [ ] Slider live SR feedback (or documented tradeoff)
- [ ] StandaloneSkeletonBox short-circuits under reducedMotion
- [ ] Animation direction semantics clarified
- [ ] IngredientsStep inputRefs resolved (wire or delete)
